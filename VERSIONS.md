# Что значат статусы?

1. _В разработке_ - версия разрабатывается
2. _Не поддерживается_ - версия не имеет обновлений безопасности
3. _Актуальная_ - Самая новая
4. _Неактульная; с обновлениями_ - Устаревшая, но с обновлениями
